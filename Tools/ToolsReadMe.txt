This folder contains the essential tools that are required for proper working in engine.

included tools:
1. CubeMapGen : a tool to create cube maps and pre filter them like 	creating irradiance map.
2. EasyChannelPacking : is a useful tool that multiple textures in to R-G-B
 channels in a texture.

These tools are not "mine" but they are open source and they didn't contain any licence.
